Getting Started with LIGO


0. Sign up for a LIGO.ORG account
Go to https://my.ligo.org.
If you are already signed in, sign out.
Click on "LSC or Virgo Signup".
Choose membership in "ACIGA - Monash University".

2. Instructions for Getting Started on the LDG: link

2. Install LDG
Install the LDG Client which enables gsissh necessary for connecting to the LIGO Data Grid computers. Follow these instructions:
  https://www.lsc-group.phys.uwm.edu/lscdatagrid/doc/installclient-mac.html
It's good to double-check this link by googling "install LIGO data grid" since the instructions are constantly changing and the web page can move.

3. Request sgwb ilog and sgwb svn accounts
Send email to ldas_admin_cit{AT}ligo.caltech.edu and CC Fred Donovan and ask for access to sgwb. Matapps access comes automatically with LIGO.ORG credentials.
Check out the stochastic and matapps svn repositories:
svn --username eric.thrane co https://svn.ligo.caltech.edu/svn/stochastic/
svn --username eric.thrane co https://svn.ligo.caltech.edu/svn/matapps/packages/stochastic/trunk

4. Log on to the LIGO Data Grid:
# First, put these commands into your .bashrc file:
proxy () { ligo-proxy-init albert.einstein; }
pcdev1=your_username@ldas-pcdev1.ligo.caltech.edu
pcdev1 () { gsissh -YC $pcdev1 ; }
# To log on, type proxy. You will be prompted for your password. Once your proxy certificate is created, you can log in by typing pcdev1. If you don't have an account on pcdev1, you can request one here.

5. Create your own web page to share results:
On pcdev1, mkdir ~/public_html/
Any content you put in public_html/ will be visible here: https://ldas-jobs.ligo.caltech.edu/~your_username.

6. Sign up to the stochastic mailing list:

7. Copy my startup.m file from ~ethrane to your home directory.
This way, matlab knows about all the necessary routines.

8. Source my .bash_login file to point your matlab to /ligotools/.
This step should eventually be replaced with your own bash_login file.
